Detailed Description

Classe que representa um escultor 3D.

O escultor é responsável por criar e manipular uma matriz tridimensional de voxels. Ele oferece métodos para realizar operações como colocar e remover voxels, criar caixas, esferas e elipsoides, definir cores e transparências, e salvar a escultura em um arquivo OFF.
